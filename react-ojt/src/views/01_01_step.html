width : 고정 / height: 고정

Inline-block

Inline-block 사용 시 필수 적용 속성 :

- 부모요소에 font-size : 0;

Table

Table 사용 시 필수 적용 속성 :

- 부모요소에 display : table;
- 해당 요소에 display : table-cell;

*** table-layout : fixed를 적용해주면 내용과 상관없이 너비가 고정되어 셀에 지정한 너비 기준으로 적용된다

Float

Float 사용 시 필수 적용 속성 :

- 부모요소에 overflow : hidden or 가상태그(확실한 방법)로 해제 해줘야 한다

- 해당 요소에 display : float;

Flex

Flex 사용 시 필수 적용 속성 :

- 부모요소에 display : flex;

Grid

Grid 사용 시 필수 적용 속성 :

- 부모요소에 display : grid;

Position

position 사용 시 필수 적용 속성 :

- 배치 기준이 되는 상위요소에 position : relative

- 해당 요소에 position : absolute 후 좌표값 지정해줘야함 *** 부모요소의 여백만큼 좌표값을 지정해줘야함
